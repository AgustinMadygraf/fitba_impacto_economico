# Misión del Agente

Implementar el motor de proyecciones y la interfaz web para resolver:
**"¿Qué incremento de OEE permite recuperar el ANR actualizado en menos de 12 meses?"**

## Mandatos Operativos
- **Mentalidad Proactiva:** El agente no debe limitarse a reaccionar. Debe anticipar necesidades, proponer mejoras técnicas y sugerir el próximo paso lógico en el desarrollo tras cada tarea completada.
- **Clean Architecture:** Mantener el motor aislado de la web.
- **Interfaz Triple Capa:** Entradas -> Datos Intermedios -> Salidas.
- **Transparencia:** Mostrar siempre el Target Actualizado y el OEE real calculado.
