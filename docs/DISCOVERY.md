# DISCOVERY - Estado de Definiciones

## 1. Mapeo Matemático (Resuelto)
El beneficio económico se calcula como:
`Beneficio = (Volumen_Objetivo - Volumen_Base) * Margen_Contribución_Unitario`
Donde el `Volumen_Objetivo` es función del incremento de Disponibilidad (D), limitado por el `Factor_Demanda` del escenario.
